# Documentation

This folder contains important project documentation.

## Files

- **DEPLOYMENT_GUIDE.md** - How to deploy and sync data to production
- **CLOUDINARY_CREDENTIALS.md** - Cloudinary setup and configuration
- **AGENT_DATABASE_SAFETY_RULES.md** - Database safety guidelines for AI agents

## Scripts

All utility scripts are in `/scripts/`:
- `restore-tag-system.js` - Restore the 14 rainbow category tags
- `fix-all-colors.js` - Fix all tag colors to match categories
- `delete-placeholder-videos.js` - Remove demo/placeholder videos
- `cleanup-extra-categories.js` - Remove extra category tags
- `test-db-persistence.js` - Test if database persists writes
